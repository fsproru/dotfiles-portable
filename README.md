#dotfiles-portable
Potrable set of UNIX configuration files

## Install
```
git clone git@github.com:fsproru/dotfiles-portable.git ~/.dotfiles
cd ~/.dotfiles
./install.sh
```

## Uninstall
```
cd ~/.dotfiles
./uninstall.sh
```
